ナレッジベース（作成中）

1. ファイルの作り方 & 名前の付け方

ファイル名は **半角英数字 & 小文字** にする。

❌ NG: AboutUs.HTML, main-STYLE.CSS
✅ OK: about-us.html, main-style.css

2. HTMLの書き方ルール

インデントは **TABキー** で統一する。

❌ NG:
<div>
<p>こんにちは</p>
</div>

✅ OK:
<div>
    <p>こんにちは</p>
</div>

3. CSSの書き方ルール

クラス名は **意味がわかる名前** にする。

❌ NG:
.button1 { ... }
.div-red { ... }

✅ OK:
.btn-primary { ... }
.alert-message { ... }

4. フォルダ構成のルール

フォルダは **役割ごとに分ける**。

/project-root/
│── index.html
│── /css/
│   │── styles.css
│── /images/
│── /js/
│   │── main.js

5. よくあるミスと対策

リンクは **相対パス** を使う。

❌ NG:
<img src="C:\Users\MyPC\Desktop\image.jpg">

✅ OK:
<img src="images/photo.jpg">
