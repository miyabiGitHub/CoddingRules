ナレッジベース

1. ファイルの作り方 & 名前の付け方

ファイル名は **半角英数字 & 小文字** にする。

❌ NG: AboutUs.HTML, main-STYLE.CSS
✅ OK: about-us.html, main-style.css

2. HTMLの書き方ルール

インデントは **TABキー** で統一する。

❌ NG:
<div>
<p>こんにちは</p>
</div>

✅ OK:
<div>
    <p>こんにちは</p>
</div>

3. CSSの書き方ルール

クラス名は **意味がわかる名前** にする。

❌ NG:
.button1 { ... }
.div-red { ... }

✅ OK:
.btn-primary { ... }
.alert-message { ... }

4. フォルダ構成のルール

フォルダは **役割ごとに分ける**。

/project-root/
│── index.html
│── /css/
│   │── styles.css
│── /images/
│── /js/
│   │── main.js

5. よくあるミスと対策

リンクは **相対パス** を使う。

❌ NG:
<img src="C:\Users\MyPC\Desktop\image.jpg">

✅ OK:
<img src="images/photo.jpg">

2.よく使うタブ

⚫️ h1~h6タグ ⚫️

見出しを作成する際に使用します。

📌 h1タグ（ Wordでいう「見出し1」）
    <h1>見出し1</h1>
📌 h2タグ（ Wordでいう「見出し2」）
    <h2>見出し2</h2>
※「h3」「h4」「h5」「h6」タグも同様

⚫️ pタグ ⚫️

テキストの段落を作成する先に使用します。

※改行目的には使わない！改行はbrタグを使う！

📌 pタグで段落作成
    <p>段落作成（ただし改行目的には使用しない）</p>

⚫️ aタグ ⚫️

リンクを埋め込む際に使用します

📌 href属性にリンク先を指定する
    <a href="https://~~~~">ここをクリックするとhrefで指定したリンクに遷移<a>
📌 target属性を追加して別タブで開くかどうか指定する
    <a href="https://~~~~" target="_blank">リンク先のページを別タブで開く<a>

⚫️ ulタグ, olタグ, liタグ ⚫️

箇条書きしたい際、リストを作成したい際に使用します

📌 ulタグとliタグで順序のないリストを作成
    <ul>
        <li>ここに内容を入力</li>
        <li>ulタグの中にliタグを記述する</li>
    <ul>
📌 olタグとliタグで順序のあるリストを作成
    <ul>
        <li>ここに内容を入力</li>
        <li>olタグの中にliタグを記述する</li>
    <ul>

⚫️ aタグ ⚫️

画像を表示させる際に使用します

📌 src属性でファイル名を指定
    <img src="sample.jpg" />
    ※imgタグは空要素と呼ばれ、要素の中身がないタグ。空要素は上記のように記述できる。

⚫️ sectionタグ ⚫️

見出しを含む文書をグループ化する際に使用します

📌 sectionでh2を含む文書をグループ化
    <section>
        <h2></h2>
        <p>段落作成</p>
    </section>
    ※h1~h6を含む必要がある

⚫️ divタグ ⚫️

制限なくグループ化する際に使用します

📌 imgタグもグループ化できる
    <div>
        <img src="sample.jpg" />
    <div>
